*******************************************************************************

1. Требования к серверному программному обеспечению

  PHP версия 5.2.4 или выше.
  MySQL версия 5.0 или выше.
  Apache/nGinx

-------------------------------------------------------------------------------

2. Рекомендации к серверному программному обеспечению

  Установленный mod_rewrite для Apache, либо подобный модуль для другого веб-сервера

-------------------------------------------------------------------------------

3. Установка

  a) Скопировать файлы в директорию веб-сервера
  b) Загрузить в базу SQL схемы /install/scheme/*_scheme.sql
  c) Настроить доступ к базе данных /app/config/database.php
  g) Установить пути к папкам в /index.php

-------------------------------------------------------------------------------

4. Обновление

  Вы всегда можете получить последнюю версию отсюда с нашей страницы на GitHub https://github.com/sharapovweb/exidoengine

-------------------------------------------------------------------------------

5. Другая информация

  ExidoEngine Web-sites manager еще не имеет публичной версии и находится в разработке
